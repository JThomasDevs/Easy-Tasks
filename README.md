# Easy Tasks

A [RuneLite](https://runelite.net/) plugin that assists with **birdhouse runs** on Fossil Island in Old School RuneScape.

## Features

- **Birdhouse highlights** — Highlights birdhouse spots by state (empty slot, empty birdhouse, occupied birdhouse) with configurable colors
- **Teleport highlights** — Highlights navigation teleports (e.g. Digsite pendant, Spirit tree) for quick travel
- **Step guidance** — Infobox and overlay showing the current step in your birdhouse run
- **Item tracking** — Infobox showing required items (logs, seeds, clockwork, etc.) so you know what to bring
- **Side panel** — Birdhouse task panel for tracking progress and steps
- **Configurable** — Highlight colors, thickness, and toggles for each overlay/infobox

## Configuration

Configure via **RuneLite → Configuration → Easy Tasks** (or search "birdhouse"):

| Option | Description |
|--------|-------------|
| **Birdhouse mode** | Loadout mode: complete birdhouses or logs + seeds |
| **Highlight birdhouses** | Toggle birdhouse object highlights |
| **Highlight teleports** | Toggle teleport object highlights |
| **Show item tracking** | Toggle item tracking infobox |
| **Show step guidance** | Toggle step guidance infobox |
| **Empty slot / birdhouse / occupied / teleport colors** | Customize highlight colors |
| **Highlight thickness** | Outline thickness (1–5) |
